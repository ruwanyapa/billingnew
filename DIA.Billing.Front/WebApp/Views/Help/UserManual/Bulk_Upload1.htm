Bulk Upload

Special Note

  • This will only facilitate to receipt that are generated after cutover (CPOS generated receipts only).

 

  1. Click on ‘Bulk Upload’ radio button. Upon selecting, ‘Correct Entry Details’ grid will be hide.

  2. Click on 'Choose File' button and select the excel file with receipt details which needs to be cancelled.

 

Special Note

  • Excel sheet will be created according to a predefined format.

 

ReceiptNO
DBPPDSC00000482
DBPPDSC00000483

 

 

  1. Click on 'Load' button. Upon clicking, data in the excel will be loaded to the ‘Cancelling Receipt Details’ grid.

 

Special Notes:

  • If user has uploaded invalid receipt numbers, then those will be hi-lighted and shown. User can't edit the number, excel have to be edited.

 

  1. Select / key-in details such as Cancellation Reason code / description, Mistake done by and cancellation remarks.

  2. If necessary, attach file to that particular batch and select the types of files that are attaching.
