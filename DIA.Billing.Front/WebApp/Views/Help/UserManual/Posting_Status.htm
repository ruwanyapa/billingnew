Posting Status

  1. User has to load the already saved details (i.e the Batch) via finder popup
  2. Then relevant receipt details will be loaded to the Receipt Details Tab.
  3. Click on ‘Posting Status’ button.
  4. Inquiry UI will be loaded and details required to query that particular receipt will be passed to that UI.
  5. User can now click on 'Search in Billing System / Search in PE / Search in CPOS' and find the posting status of that receipt.
