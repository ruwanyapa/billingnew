Bill Invoice Number

  1. Select the BU and Product Category (Default value for Product Category is 'CCBS' and for BU is ‘All’)

  2. Select the ‘Bill Invoice Number' radio button and then key-in the relevant number

  3. to the text field and press Enter key.

  4. System will automatically load relevant customer’s account details to the grid

  5. Then user can select the relevant numbers he/she wants to submit and then key-in  the paying amount to the ‘Pay Amount’ field.

  6. If user wants he/she can retrieve more details from RBM, PE, OCS and Other systems by clicking on respective line-wise button (to retrieve details of one number) or bulk buttons (to retrieve details for all the numbers).

  7. If user wants he/she will be able to attach a file or add remarks to that particular batch he/she is creating.  

 

Special Notes: If user adds a remark to the Remark field in ‘batch summary’ widget, then that same remark will be reflected in the Remark column in the grid. However if user wants to add a custom remark to one record he/she will be able to edit that respective cell and add a custom remark.
