Query from CPOS

Special Note

 

    • Payments which are failed posting to PE (Stuck in CPOS) can be queried from PE and found the required details.

 

 

  1. User shall enter the required parameters that need to inquire a receipt/s. Following are the search combinations user can use to perform a search. No other search combinations will be facilitated.

 

    • Product Category + Receipt branch + receipt no

    • Product Category + Receipt branch + payment method

    • Product Category + Receipt Branch + from date + to date

    • Product Category + Cheque no

    • Product Category + Bank code + branch code + chq no

    • Product Category + Contract Number + from date + to date

    • Product Category + Receipt Branch + from date + to date + Receipt User

    • Product Category + From date + To date + Receipt User

 

 

  1. Click on ‘Search in CPOS’ button.

  2. Then the relevant details will be loaded to the grid.
