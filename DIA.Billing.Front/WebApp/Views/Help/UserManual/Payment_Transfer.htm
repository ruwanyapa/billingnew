Payment Transfer

 

  1. Find the receipt/s details which needs to Transfer.
    • User can search the relevant receipt/s using ‘Bulk Payment’ UI or ‘Payment Inquiry’ UI.
    • Click on line-wise 'Transfer' button.

 

  1. Relevant receipt details will be loaded to the 'Cancelling Receipt Details' grid in the Payment Transfers GUI.
  2. Select / key-in details such as Transfer Reason code / description, Mistake done by, Customer Contact details and Transfer remarks.
  3. If necessary, attach file to that particular batch and select the types of files that are attaching.

 

Special Note

  • For some users attaching documents is compulsory. Refer following diagram for more info.

Special Notes:

  • If user has keyed-in an invalid number, then those will be hi-lighted and shown. User can edit the number and validate it again by clicking on CRM button.
