Batch Cancellation

  1. Click on ‘Batch Cancellation’ radio button. Upon selecting, ‘Correct Entry Details’ grid will be hide.

  2. Click on 'Find' button and select the CPOS generated Batch which needs to be cancelled.

  1. Relevant data will be loaded to the ‘Cancelling Receipt Details’ grid.

  1. Select / key-in details such as Cancellation Reason code / description, Mistake done by and cancellation remarks.

  2. If necessary, attach file to that particular batch and select the types of files that are attaching.

 

 

Special Note

  • For some users attaching documents is compulsory. Refer following diagram for more info.
