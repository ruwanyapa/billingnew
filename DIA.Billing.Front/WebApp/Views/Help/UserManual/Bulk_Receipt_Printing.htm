Bulk Receipt Printing

  2. Click on ‘Browse’ button and find the excel which needs to upload.

 

 

Special Note

  • Excel sheet will be created according to a predefined format.

  1. Click on ‘Load’ button.  
  2. Once clicked, data in the excel sheet will be loaded to the grid along with the other related data.

 

 

Special Notes:

  • If user has uploaded invalid receipt numbers, then those will be hi-lighted and shown. User will not be able to print them.
